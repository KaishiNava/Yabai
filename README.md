# Yabai Famss

“Yabai Famss bukan tentang siapa yang paling terkenal,
bukan juga tentang siapa yang paling lama bertahan.

Kadang circle terbaik lahir dari orang-orang random
yang dipertemukan oleh kenyamanan sederhana.

Di sini kita ngobrol, bercanda, saling ganggu,
dan tanpa sadar jadi bagian dari cerita satu sama lain.

Kalau suatu saat nanti member berubah,
kesibukan datang,
atau chat mulai sepi,
setidaknya Yabai pernah jadi tempat
yang membuat seseorang merasa tidak sendirian.”
